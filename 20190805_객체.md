# 객체
## 객체란

- 자바스크립트에서는 객체가 왕이다. 거의 모든 것들이 객체이거나 객체처럼 동작함
- 
## 객체 생성방법

```javascript
var jb = { 'a':100, 'b':200, 'c':300 };
```
```javascript
var jb = {};
jb['a'] = 100;
jb['b'] = 200;
jb['c'] = 300;
```

```javascript
var jb = new Object();
jb['a'] = 100;
jb['b'] = 200;
jb['c'] = 300;
```

```javascript
var jb = new Object();
jb.a = 100;
jb.b = 200;
jb.c = 300;
```
